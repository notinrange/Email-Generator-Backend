
# Mailer Project Backend How to Run Locally

- **Step 1** Clone repository
```
git clone https://github.com/notinrange/Email-Generator-Backend.git
```

- **Step 2** Run jre file
```
java -jre 
```

for frontend follow readme file steps of this repository
